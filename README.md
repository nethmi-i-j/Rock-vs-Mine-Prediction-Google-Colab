# Rock-vs-Mine-Prediction-Google-Colab

A Machine Learning project that predicts whether an underwater object is a Rock or a Mine using sonar signal data. This project uses classification algorithms to analyze sonar readings and identify objects accurately.

## Project Overview

This project focuses on predicting whether an underwater object is a rock or a mine using sonar signal data by analyzing the reflected sonar waves received from different underwater objects and identifying patterns in the data to classify the object accurately.

## Dataset Information

* Dataset: Sonar Dataset from kagle
* Features: 60 numerical sonar signal attributes
* Number of columns: 208
* Target:

     R ---> Rock

     M ----> Mine

## Model Used

**Logistic Regression**
